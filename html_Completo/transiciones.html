Transiciones en CSS

Las transiciones en CSS permiten cambiar propiedades de un elemento de forma suave en un periodo de tiempo. Se usan principalmente con eventos como :hover.

Propiedades principales:
transition-property: propiedad a animar
transition-duration: duración
transition-timing-function: velocidad de animación
transition-delay: retraso

Tipos de velocidad

Las funciones de tiempo controlan cómo se mueve la animación:


ease: suave (por defecto)
linear: velocidad constante
ease-in: empieza lento
ease-out: termina lento
ease-in-out: inicio y final lento
