Cajas (box model)

En html y css, todos los elementos se representan como cajas rectangulares, esto se conoce como el modelo de caja (box model). El modelo de caja es un concepto fundamental en CSS que describe cómo se calculan las dimensiones y el espacio ocupado por los elementos en una página web. Cada elemento se compone de cuatro partes principales:

El modelo de cajas esta compuesto por:
Contenido: Es el área donde se muestra el contenido del elemento (texto, imágenes o cualquier otro tipo de contenido)
Relleno (padding): Es el espacio entre el contenido y el borde de la caja
Borde (border): borde que rodea la caja
Margen (margin): Espacio externo entre elementos

propiedades de las cajas

Las cajas pueden modificarse usando sus diferentes propiedades


width: ancho
height: alto
padding: espacio interno
margin: espacio externo
border: borde

box-sizing

la propiedad box-sizing permite controlar como se calculan las dimensiones de las cajas


content-box: es el valor por defecto, el ancho y alto se aplican solo al contenido, el padding y el borde se agregan al tamaño total de la caja
border-box: el ancho y alto incluyen el contenido, el padding
