Formularios en HTML

Los formularios permiten recopilar información del usuario, como nombres, correos, contraseñas, entre otros. Se crean con la etiqueta form.

Elementos principales:
input: campo de entrada
label: etiqueta descriptiva
textarea: área de texto
select: lista desplegable
button: botón

Tipos de input

Existen diferentes tipos de campos:


text: texto
email: correo
password: contraseña
number: números
date: fechas
checkbox: selección múltiple
radio: selección única

Validación básica

HTML permite validar datos automáticamente usando atributos como:


required: obligatorio
min / max: valores mínimos y máximos
maxlength: longitud máxima
