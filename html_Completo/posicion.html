Posición en CSS

La propiedad position en CSS permite controlar la ubicación de los elementos dentro de la página.

Los tipos de posición son:
static: posición por defecto
relative: se mueve respecto a su posición original
absolute: se posiciona respecto al contenedor padre
fixed: se fija en la pantalla
sticky: se comporta como relative y luego como fixed

Propiedades adicionales

Para mover elementos posicionados se usan:


top: desde arriba
bottom: desde abajo
left: desde la izquierda
right: desde la derecha
