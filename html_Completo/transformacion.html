Transformaciones en CSS

La propiedad transform en CSS permite modificar la forma, tamaño o posición de un elemento sin afectar el flujo del documento.

Los tipos de transformaciones son:
translate: mover el elemento
rotate: rotar el elemento
scale: cambiar el tamaño
skew: inclinar el elemento
