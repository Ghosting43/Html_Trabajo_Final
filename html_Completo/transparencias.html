Transparencias en CSS

Las transparencias en CSS permiten hacer que los elementos sean parcialmente visibles. Se pueden aplicar usando la propiedad opacity o colores con rgba.

Formas de aplicar transparencia:
opacity: controla la transparencia de todo el elemento
rgba: controla la transparencia del color

Diferencia importante

opacity afecta todo el elemento (incluyendo texto).
rgba solo afecta el fondo.
