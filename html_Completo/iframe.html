Iframe en HTML

El elemento iframe permite incrustar otra página web dentro de tu sitio. Es muy útil para mostrar videos, mapas, páginas externas, entre otros.

Atributos principales:
src: URL del contenido
width: ancho
height: alto
frameborder: borde del iframe
allowfullscreen: permitir pantalla completa
