web storage

web storage permite guardar información en el navegador del usuario sin necesidad de una base de datos se utilizan principalmente localstorage y sessionStorage

tipos de almacenamiento:
localStorage: almacena datos de forma permanente
sessionStorage: almacena datos de la sesion mientras la pestaña esta abierta
