Columnas en CSS

Las columnas en CSS permiten dividir el contenido en varias partes verticales, similar a un periódico. Se usan principalmente con propiedades como column-count y column-width.

Propiedades principales:
column-count: número de columnas
column-width: ancho de cada columna
column-gap: espacio entre columnas
column-rule: línea entre columnas
